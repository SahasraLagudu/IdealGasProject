Instructions on how to run the simulation: 

1) Create a new Eclipse Project Folder. 
2) Download the MainDisplay.java file from this repository and add it to the Project Folder (into the src folder). 
3) Download the gifs from this repository and copy and paste them into the src folder of the project. 
4) Go to the MainDisplay and run the simulation by clicking on the green button (enter values as prompted and explained on the User Instructions and Design Paragraph on the poster submission). (This is the process/elements of project and instructions paragraph)).
